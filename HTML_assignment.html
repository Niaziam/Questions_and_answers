Step 1: Basic HTML Questions

1. What is HTML? Explain its purpose & Basic Syntax of HTML

ANS: HTML (Hypertext Markup Language) is the standard language used to create webpages. Its purpose is to structure content on the web.

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>
<h1>Hello, World!</h1>
<p>Welcome to HTML basics.</p>
</body>
</html>

2. What are HTML tags? How are they structured? Give Us 5 Tags As A Example

HTML tags define elements in a webpage. Tags are written inside angle brackets (<>) and usually come in pairs with an opening and a closing tag. Example of tags:

  • <h1> - Defines a heading
  • <p> - Defines a paragraph
  • <a> - Defines a hyperlink
  • <img> - Defines an image
  • <ul> - Defines an unordered list

Difference Between <head>, <body>, <html> Tags & The Role of <!DOCTYPE html>

1. <html> Tag

The <html> tag is the root element of an HTML document. It wraps all the content on the webpage, including both the <head> and <body> sections.

2. <head> Tag

The <head> tag contains metadata about the webpage. Metadata includes the title, character encoding, stylesheets, and other information that isn't displayed directly on the webpage.

Example: <title>Page Title</title> defines the title displayed on the browser tab.

3. <body> Tag

The <body> tag contains the visible content of the webpage. This includes text, images, videos, and other media elements.

Example: <h1>Hello, World!</h1> is a heading visible on the page.

4. <!DOCTYPE html> Declaration

The <!DOCTYPE html> declaration is placed at the top of an HTML document. It tells the browser that the document is written in HTML5. This ensures consistent rendering of the webpage across different browsers.

4. Describe the function of the <title> tag. Where does it appear With Example

The <title> tag defines the title of the webpage, which appears in the browser tab. Example: <title>My Awesome Page</title> will display "My Awesome Page" in the tab.

5. How do you add comments in HTML, and why are they useful?

Comments in HTML are added using <!-- comment -->. They are useful for leaving notes or explanations for developers and are not visible in the browser.

Example: <!-- This is a comment -->

Step 2: HTML Elements and Attributes

1. Difference Between an HTML Element and an HTML Attribute

An HTML element is the entire structure, including the opening tag, content, and closing tag. An HTML attribute provides additional information or functionality for an element.

Example:

HTML Element: <h1>Welcome</h1>

HTML Attribute: <p style="color:red;">This is red text.</p> (Here, style is the attribute)

2. Global Attributes in HTML

Global attributes can be used on any HTML element. Examples include:

  • class: Groups elements for styling.
  • id: Provides a unique identifier for an element.
  • style: Adds inline CSS styling to an element.
  • title: Adds a tooltip when hovering over the element.

Example:

<div class="container" id="main" style="color:blue;" title="Hover text">Sample Text</div>

3. Purpose of the alt Attribute in the <img> Tag

The alt attribute provides alternative text for an image. It's displayed if the image cannot load or is read by screen readers for accessibility.

Example:

<img src="image.jpg" alt="A beautiful sunset">

4. Describe the href Attribute in the <a> Tag

The `href` attribute specifies the URL of the page the link points to. It's essential for creating hyperlinks.

Example:

<a href="https://tanimImage.com">Visit Example</a>

5. The target Attribute in the <a> Tag

The `target` attribute specifies where to open the linked document. The `_blank` value opens the link in a new tab or window.

Example:

<a href="https://tanimImage.com" target="_blank">Open in New Tab</a>

Step 3: Common HTML Tags

1. How to Add Favicon in HTML & Why?

To add a favicon, you use the <link> tag inside the <head> section. Example: <link rel="icon" href="favicon.ico">.

It helps make the website recognizable and adds a professional touch to the web page.

2. What is Formatting in HTML?

Formatting in HTML refers to the way text is displayed. It uses tags like <b> for bold, <i> for italic, and <u> for underline. Example: <b>Bold Text</b>, <i>Italic Text</i>.

3. What Are Inline and Block Elements in HTML5? Give Us the Proper Explanation with Example

Inline elements: only take up as much space as their content requires. They donâ€™t break to the next line. Example: <a>, <span>.

Block elements: take up the full width of their parent container and start on a new line. Example: <div>, <p>.

4. What is a Marquee?

The <marquee> tag makes text scroll horizontally or vertically on the screen. Example: <marquee>This is a scrolling text</marquee>.

5. Explain the Difference Between <p>, <span>, and <div> Tags. When Would You Use Each?

<p> is a block-level tag used for paragraphs. It creates a new line before and after its content. Example: <p>This is a paragraph</p>.

<span> is an inline element, typically used to style a portion of text within other elements. Example: <span style="color:red;">This is red text</span>.

<div> is a block-level container used to group elements together. It has no visual impact unless styled. Example: <div>Content here</div>.

6. Describe the <ul>, <ol>, and <li> Tags for Creating Lists

<ul>: Unordered list, typically displayed with bullets. Example: <ul><li>Item 1</li></ul>.

<ol>: Ordered list, displayed with numbers or letters. Example: <ol><li>First item</li></ol>.

<li>: List item, used inside <ul> or <ol>. Example: <li>Item 1</li>.

7. Describe How <input> Works and Some of Its Types, Such as text, radio, and checkbox

The <input> tag is used to create interactive controls in a form. Examples of input types:

  • text: For text fields. Example: <input type="text">.
  • radio: For radio buttons, allowing one selection in a group. Example: <input type="radio">.
  • checkbox: For checkboxes, allowing multiple selections. Example: <input type="checkbox">.

8. What Are <button>, <textarea>, and <select> Tags, and How Do They Function?

<button>: Defines a clickable button. Example: <button>Click Me</button>.

<textarea>: Defines a multi-line text input area. Example: <textarea>Default text</textarea>.

<select>: Defines a drop-down list. Example: <select><option>Option 1</option></select>.

9. How to Link CSS to HTML?

CSS can be linked to an HTML file using the <link> tag in the <head> section. Example: <link rel="stylesheet" href="styles.css">.

10. What is the <label> Tag, and How Is It Associated with Form Inputs?

The <label> tag is used to define labels for form elements. It improves accessibility. Example: <label for="name">Name</label><input type="text" id="name">.

Step 4: Advanced HTML Concepts

1. What is the <iframe> Tag, and What Are Some Common Attributes Used With It?

The <iframe> tag is used to embed another HTML page or external content into a webpage. Common attributes:

  • src: URL of the embedded content.
  • width: Width of the iframe.
  • height: Height of the iframe.
  • frameborder: Defines whether the border is displayed (obsolete in modern HTML).

2. Describe How the <audio> and <video> Tags Work

<audio> and <video> tags are used to embed audio and video files respectively. Common attributes:

  • controls: Adds play, pause, and volume controls.
  • autoplay: Automatically starts playback when the page loads.
  • loop: Replays the media in a loop.
Example: <audio controls><source src="audio.mp3" type="audio/mpeg"></audio> <video controls><source src="video.mp4" type="video/mp4"></video>

3. Explain the <picture> Tag and Its Usage in Responsive Design

The <picture> tag is used for responsive images, allowing multiple source images to be defined for different screen sizes or resolutions. Example: <picture> <source media="(max-width: 600px)" srcset="small.jpg"> <source media="(min-width: 601px)" srcset="large.jpg"> <img src="default.jpg" alt="Responsive Image"> </picture>
