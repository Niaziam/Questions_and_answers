Basic CSS Questions

1. What is CSS, and how does it work with HTML?

ANS: CSS (Cascading Style Sheets) is a language used to style HTML elements. It controls the layout, colors, fonts, and design of the web page. While HTML structures the content of a page, CSS makes it look good.

<style>
body {
background-color: lightgray;
}
h1 {
color: darkgreen;
}
p {
font-size: 18px;
}
</style>

2. What is the difference between inline, internal, and external CSS?

ANS: Inline CSS is written directly within an HTML element using the style attribute. Internal CSS is written inside a <style> tag within the head section of an HTML file. External CSS is stored in a separate .css file and linked to the HTML file using a <link> tag.

<!DOCTYPE html>
<html>
<head>
<style> /* Internal CSS */
body { font-family: Arial; }
</style>
<link rel="stylesheet" href="styles.css"> /* External CSS */
</head>
<body style="background-color: yellow;"> /* Inline CSS */
<h1>Hello</h1>
</body>
</html>

3. Explain the CSS box model. What properties does it consist of?

ANS: The CSS box model is a layout model that defines the space an element occupies on a page. It consists of content, padding, border, and margin.

<style>
div {
width: 200px;
padding: 10px;
border: 5px solid black;
margin: 20px;
}
</style>

4. What is specificity in CSS, and how does it affect styling?

ANS: Specificity is a calculation that determines which CSS rule takes precedence when multiple rules target the same element. It is calculated based on selectors used (inline styles > IDs > classes > elements).

<style>
#id { color: blue; }
.class { color: red; }
p { color: green; }
</style>

5. How do CSS selectors work, and what are some common types of selectors?

ANS: CSS selectors target HTML elements based on their tag, ID, class, or attributes. Common selectors include universal (*), type (h1), class (.className), ID (#idName), and attribute ([type="text"]).

<style>
* { margin: 0; }
h1 { font-size: 2em; }
.className { color: orange; }
#idName { background-color: pink; }
[type="text"] { border: 1px solid gray; }
</style>

6. What are pseudo-classes and pseudo-elements? Give examples of each.

ANS: Pseudo-classes define special states of elements (e.g., :hover). Pseudo-elements style parts of elements (e.g., ::before). Examples: :hover for link hover effect, ::before to add content.

<style>
a:hover { color: red; }
p::before { content: "Read: "; }
</style>

7. Explain the concept of inheritance in CSS.

ANS: Inheritance in CSS means certain properties (e.g., font-family, color) are automatically passed down from parent elements to child elements.

<style>
body { font-family: Arial; color: navy; }
p { margin: 10px; }
</style>

8. What is the `!important` rule in CSS, and when should you use it?

ANS: The `!important` rule overrides all other specificity rules, including inline styles. Use it sparingly to avoid making code difficult to maintain.

<style>
p { color: red !important; }
p { color: green; } /* The color will still be red because of !important */
</style>

Mostly Used CSS Properties

1. How does the display property work, and what are the different display values?

ANS: The display property defines how an element is displayed on the web page. Common values include block (full width), inline (inline with other elements), flex (flexbox layout), and none (hides element).

div {
display: flex;
justify-content: center;
}
span {
display: inline;
}

2. What is the position property, and how do different position values behave?

ANS: The position property defines how an element is placed on a page. static (default), relative (relative to normal position), absolute (relative to nearest positioned ancestor), fixed (relative to viewport), and sticky (toggles between relative and fixed based on scroll).

div {
position: fixed;
top: 0;
left: 0;
}

3. How do the padding, margin, and border properties work in CSS?

ANS: padding defines space between content and border. margin defines space outside the border. border defines the edge around an element.

div {
padding: 20px;
margin: 15px;
border: 2px solid black;
}

4. Explain the flex property. How do flex-grow, flex-shrink, and flex-basis work?

ANS: The flex property defines how flexible an item is within a flex container. flex-grow defines how much it grows relative to other items. flex-shrink defines how it shrinks. flex-basis defines the initial size.

div {
display: flex;
}
.item {
flex: 1 1 auto;
}

5. What is the difference between width, min-width, and max-width?

ANS: width sets an element's fixed width. min-width defines the minimum width an element can shrink to. max-width defines the maximum width an element can expand to.

div {
width: 300px;
min-width: 200px;
max-width: 400px;
}

6. How does the z-index property work, and when would you use it?

ANS: The z-index property defines the stacking order of elements. Higher values are on top. Used for overlapping elements.

div {
position: absolute;
z-index: 10;
}

7. What is the difference between the opacity and visibility properties?

ANS: opacity controls transparency (0 = invisible, 1 = fully visible). visibility controls whether an element is visible or hidden (it still takes space when hidden).

div {
opacity: 0.5;
visibility: hidden;
}

CSS Layouts & Responsiveness

1. What is the difference between flexbox and CSS Grid, and when should you use each?

ANS: flexbox is one-dimensional (row or column), great for aligning items. CSS Grid is two-dimensional (rows and columns), ideal for complex layouts. Use flexbox for navbars or simple alignments, and grid for full page layouts.

Flexbox:
div {
display: flex;
}
Grid:
div {
display: grid;
grid-template-columns: repeat(3, 1fr);
}

2. Explain how to create a responsive layout using media queries.

ANS: Media queries allow you to apply styles based on screen size or device features.

@media (max-width: 600px) {
div {
font-size: 14px;
}
}

3. How do you create a centered element in CSS? Provide multiple ways.

ANS: Centering options: flexbox, grid, margin, or positioning.

Flexbox:
div {
display: flex;
justify-content: center;
align-items: center;
}
Grid:
div {
display: grid;
place-items: center;
}
Margin:
div {
margin: auto;
}

4. How does CSS Grid work, and what are grid-template-rows and grid-template-columns?

ANS: CSS Grid defines layouts using rows and columns. grid-template-rows and grid-template-columns define the sizes of rows and columns.

div {
display: grid;
grid-template-rows: 100px auto;
grid-template-columns: 1fr 2fr;
}

5. How does the flex-direction property work in Flexbox, and what are its values?

ANS: flex-direction defines the direction of flex items. Values: row, row-reverse, column, column-reverse.

div {
display: flex;
flex-direction: column;
}

6. What is a viewport, and how does it affect the responsiveness of a website?

ANS: A viewport is the visible area of a web page. Relative units like %, vw, and vh help maintain responsiveness across devices. Use media queries to adapt layout to screen sizes.

7. How would you create a sticky footer?

ANS: You can use flexbox or positioning to keep a footer at the bottom of the page.

8. What is mobile-first design in CSS?

ANS: Mobile-first design starts with styles optimized for small screens. Use media queries to adjust styles for larger screens.

Advanced CSS Questions

1. What do you understand by the universal selector?

ANS: The universal selector (*) selects all elements on a page, applying styles globally.

* {
margin: 0;
padding: 0;
box-sizing: border-box;
}

2. What are CSS preprocessors, and how do they work? Give examples like SASS or LESS.

ANS: CSS preprocessors like SASS or LESS allow writing CSS with variables, nested rules, and functions. They compile into regular CSS.

/* SASS Example */
$primary-color: #3498db;
body {
background-color: $primary-color;
}

3. Explain CSS custom properties (CSS variables) and their benefits.

ANS: CSS variables start with -- and can be reused throughout a style sheet. They make CSS maintainable and dynamic.

:root {
--main-color: #f39c12;
}
div {
color: var(--main-color);
}

4. How do CSS animations work, and how can you create a keyframe animation?

ANS: CSS animations use @keyframes to define animated states. Assign the animation using animation-name and animation-duration.

@keyframes slide {
0% { transform: translateX(0); }
100% { transform: translateX(100px); }
}
div {
animation: slide 2s infinite;
}

5. What are CSS transitions, and how are they different from animations?

ANS: Transitions smoothly animate changes between states triggered by events. Animations run continuously or based on timing.

/* Transition Example */
div {
transition: background-color 0.3s ease;
}
div:hover {
background-color: #3498db;
}

6. What is the difference between rem, em, and px units?

ANS: px: Absolute unit. rem: Relative to root font size. em: Relative to parent element's font size.

html { font-size: 16px; }
div { font-size: 2rem; } /* 32px */
p { font-size: 1.5em; } /* 1.5 times the parent size */

7. What is SVG, and how can it be used to create custom shapes?

ANS: SVG (Scalable Vector Graphics) is an XML format for drawing shapes. It's resolution-independent and can be inline in HTML or as files.

<svg width="100" height="100">
<circle cx="50" cy="50" r="40" fill="blue" />
</svg>
